# C# Variables

## What type of variables are there in C#?
### There are 6 types of variables in C#
- var
- string
- int
- const
- double
- bool

## Var Example(new):
```c
var name = "Dayne"; or var num = 0;
```
## String Example:
``` c
string color = "red";
```
## Int Example:
```c
int count = 10;
```
## Const Example:
```c
const score = 24;
```
## Double Example:
```c
double currency = 13;
```
## Bool Example:
```c
bool keepgoing = true;
```
